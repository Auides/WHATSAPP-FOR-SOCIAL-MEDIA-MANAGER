# Mobile app

This is an Expo-based mobile app that loads the backend setup/manager portal in a WebView.

## Quick start

1) Install dependencies:

```
npm install
```

2) Start the app:

```
npm run start
```

3) Enter your backend URL in the app (use your machine's LAN IP, not localhost).

Example:

```
http://192.168.1.10:3000/setup
```
